## bedgraph
首先来介绍下bedgraph格式，这种格式实际上就是用窗口的方式代替原始的每个碱基的测序深度，文件内容可以分为两个部分
1. track  line
2. data line
track line是首行的一句声明，内容如下
```
track type=bedGraph
```
