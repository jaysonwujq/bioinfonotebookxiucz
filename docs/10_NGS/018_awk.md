## bash group by word and concat
```
tiger sherry
tiger lina
tiger piku
tiger tery
cat kate
cat Gatsby

 awk '
BEGIN{
  OFS=","
}
{
  a[$1]=(a[$1]?a[$1] OFS:"")$2
}
END{
  for(i in a){
    print i FS a[i]
  }
}
' 11 > ven
```
