```
library(IRanges)
ir <- IRanges(5,10)
```

# GRanges
GRanges对象包含了Iranges，它含有2个方面的重要信息：
+ 染色体信息（Bioconductor称为 seqnames）。
+ DNA链。
```
seqnames(gr) #查看基因的名称
ranges(gr) #查看基因的区域
strand(gr) #查看基因位于那条链
start(gr)
end(gr)
width(gr)
names(gr)
length(gr)
seqinfo(rg)@seqnames

```
